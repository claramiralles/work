Rock & Code

Front-End development

Technical summary:

Website built in HTML & CSS (use of SASS as CSS pre-processor), following responsive design principles. Use of flexbox and grid as web layout systems. Built by a 5-developers-team, following agile principles & scrum methodologies.

Usage:

Showcasing the first development team created at our front-end training. Describes the team in different scrollable sections, including the team mission, its strengths and weaknesses and a personalized space for each team member. Contact section with form using formspread.

Features:

- Responsive website
- One scrollable page
- Navigation menu
- Form to contact with the team
